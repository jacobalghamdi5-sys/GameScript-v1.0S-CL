# Contributing to GameScript

Welcome! We are excited to scale GameScript into a massive framework. 

## How to Suggest Features
1. Look at our Roadmap in the README to see if your idea aligns with upcoming modes (like the Action engine planned for future versions).
2. Open a public Issue on this repository titled `[Feature Proposal] Your Idea`.
3. Provide a clear example of your proposed syntax using the native structural dividers (`___`).

## Code of Conduct
Be respectful, keep syntax clean, and ensure all submissions respect our Apache License 2.0 parameters.
